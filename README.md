# PrepStreak - DSA Learning Platform

A modern, gamified DSA (Data Structures & Algorithms) learning platform built with React, Vite, and Tailwind CSS.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Features

- **Gamified Learning** - Tackle complex topics through engaging challenges
- **5-min Daily Quizzes** - Build consistent practice habits
- **AI-Powered Hints** - Get intelligent support when stuck
- **Responsive Design** - Beautiful UI with Tailwind CSS
- **Modern Tech Stack** - React 18, Vite, React Router

## Project Structure

```
src/
├── App.jsx              # Main app with routing
├── main.jsx             # Entry point
├── index.css            # Global styles
└── screens/
    └── Frames/
        ├── Frame1/      # Landing page
        ├── Frame2/      # Onboarding
        └── Frame3/      # Dashboard with stats & tracking
```

## Routes

- `/` - Landing page with features showcase
- `/get-started` - Onboarding and learning path personalization
- `/dashboard` - User dashboard with performance stats and progress tracking
